AlphaRed7 GUI

Release Notes

v0.0.3

Improvements

  • 设置了应用图标。
  • 提升与段老师平台输入时的兼容性。
  • 去掉无用save settings按钮，改成自动保存，并且现在真的能save了。
  • 比赛界面左侧添加了颜色条。
  • 支持对手牌排序(默认按颜色，可选按数字)。

Fixes

  • 本地人机对战时不显示自己用户名。

v0.0.2

Fixes

  • Mac版删除了多余的调试信息，防止Segmentfault崩溃。

v0.0.1

  • 第一个公开版本!

Downloads

校内免流~

  • Windows: AlphaRed7-GUI-0.0.3-win-x86.zip
  • macOS: AlphaRed7-GUI-0.0.3-mac.zip
  • Linux: AlphaRed7-GUI-0.0.3-linux-x64.tar.gz
macOS用户建议使用新版程序直接替换旧版程序，防止丢失用户信息。

Future Features

  • 支持重置本地设置。
  • 观战功能。
  • 导出对战记录与播放对战记录。
  • 多AI辅助人机对战。
  • 用户信息导入与导出。
  • 支持双AI无动画多局对战，方便观察两AI胜率。
  • 其他建议或bug可以微信私聊或使用软件右下角提交bug按钮。
